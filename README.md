# ⚛️ CERN-Inspired-Physics-Simulator
An interactive particle physics simulator featuring Standard Model visualization, particle decay, Higgs potential, neutrino oscillation, relativity effects, and cosmic ray air shower simulations using Python and Matplotlib.

## 🚀 Features

### ⚛ Standard Model Visualization
- Particle mass spectrum
- Fundamental interactions comparison
- Log-scale physics plots

### ⏳ Particle Decay
- Exponential decay simulation
- Multiple particle lifetimes
- Scientific visualization

### 💚 Higgs Potential
- Mexican Hat Potential
- Symmetry breaking visualization

### 🌊 Neutrino Oscillation
- Oscillation probability plots
- Distance-dependent transitions

### 🚀 Relativity
- Lorentz gamma factor
- Time dilation visualization

### ☄ Cosmic Ray Air Shower
- Secondary particle generation
- Atmospheric shower simulation
- Ground particle detection
